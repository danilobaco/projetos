## Como Usar

### Método 1: Clicar no Ícone 🔗
Clique no ícone 🔗 ao lado do título de qualquer projeto. O link será copiado automaticamente para seu clipboard!

### Método 2: URL Direta
Use a URL base + âncora:
```
https://seu-dominio.com/index.html#xai-timelime
```

Ou compartilhe direto:
```
index.html#senview
```

### Método 3: Usar os IDs da Tabela
Copie o ID do projeto desejado da tabela acima e adicione `#` na frente.

## Exemplo de Compartilhamento

Para compartilhar o projeto XAI:
- Link: `https://seu-dominio.com/index.html#xai-timelime`
- Ou simplesmente: `#xai-timelime` (no mesmo domínio)

Quando alguém abrir o link, a página rolará automaticamente direto para o projeto!

**Dica:** Você pode editar os IDs em `index.html` para valores mais personalizados se desejar.
